# Core Skills Drill 5B — Tree-Based Model Basics

Module 5 Week B drill for AI.SPIRE Applied AI & ML Systems.

## Setup

```bash
pip install -r requirements.txt
```

## Tasks

Complete the three functions in `drill.py`:
1. `train_decision_tree` — Train a DecisionTreeClassifier
2. `get_feature_importances` — Extract sorted feature importances
3. `train_balanced_forest` — Train a RandomForest with balanced class weights

## Submission

1. Create a branch: `drill-5b-tree-basics`
2. Complete `drill.py`
3. Open a PR to `main`
4. Paste your PR URL into TalentLMS → Module 5 Week B → Drill 5B
